## 技術特色

### 字串建構技術
```mips
# 示例：建構 "Hi" 字串
addi $t0, $zero, 0x48       # 'H' = 0x48
addi $t1, $zero, 0x69       # 'i' = 0x69
sll $t1, $t1, 8            # 將 'i' 移到位元組位置 1
add $t0, $t0, $t1          # 組合 H + i = "Hi\0\0"
sw $t0, 0x1000($zero)      # 存儲到記憶體位址 0x1000
```

### SLL 指令驗證
- 確保 SLL（Shift Left Logical）指令正確實作
- 驗證位元移位操作：0x69 << 8 = 0x6900
- 這是字串建構的關鍵指令

### 測試覆蓋
- **記憶體操作**: 驗證字串正確存儲到指定位址
- **系統呼叫**: 測試 syscall 4 (print_string) 和 syscall 1 (print_int)
- **程式流程**: 確保程式正常執行並正確結束
- **輸出驗證**: 檢查控制台輸出包含預期內容
