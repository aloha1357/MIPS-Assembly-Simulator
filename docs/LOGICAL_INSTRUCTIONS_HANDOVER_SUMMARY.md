## 🎮 Demo Program Usage

The demo program `asmtest/logical_instructions_demo.asm` demonstrates:

1. **Basic Logical Operations**
   - AND: `0xAAAA & 0x5555 = 0x0000`
   - OR: `0xAAAA | 0x5555 = 0xFFFF`  
   - XOR: `0xAAAA ^ 0x5555 = 0xFFFF`
   - NOR: `~(0xAAAA | 0x5555) = 0xFFFF0000`

2. **XOR Encryption/Decryption**
   - Original: `0x1234`
   - Encrypted: `0x1234 ^ 0xABCD = 0xB9F9`
   - Decrypted: `0xB9F9 ^ 0xABCD = 0x1234` ✅

**Expected Console Output:**
```
AND result: 0
OR result: 65535
XOR result: 65535
NOR result: -65536

Decrypted: 4660
```
